HTML Tables

Html's Table: Tags and Description
Tag Definition
<table> The <table> element has content that is used to represent a two-dimensional table made of rows and columns.
<tr> The table row element, <tr>, is used to add rows to a table before adding table data and table headings.
<td> The Table data element, <td>, can be nested inside a table row element, <tr>, to add a cell of data to a table.
<tbody> The table body element, <tbody>, is a semantic element that will contain all tabel data other than table heading and table footer content. If used, <tbody> will contain all table row <tr> elements, and indicates that <tr> elements make up the body of the table. <table> cannot have both <tbody> and <tr> as direct children.
<thead> The table head element, <thead>, defines the headings of table columns encapsulated in table rows.
<tfoot> The table footer element, <tfoot>, uses table rows to give footer content or to summarize content at the end of a table.
<th> The table heading element, <th>, is used to add titles to rows and columns of a table and must be enclosed in a table row element, <tr>.

Table's Attributes
Attribute Definition
colspan The colspan attribute on a table header <th> or table data <td> element indicates how many columns that particular cell should span within the table. The colspan value is set to 1 by default and will take any positive integer between 1 and 1000.
rowspan Similar to colspan, the rowspan attribute on a table header or table data element indicates how many rows that particular cell should span within the table. The rowspan value is set to 1 by default and will take any positive integer up to 65534.
